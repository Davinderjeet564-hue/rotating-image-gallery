# Rotating Image Gallery

A 3D rotating carousel gallery built with vanilla HTML, CSS, and JavaScript.

## Features

- 3D carousel with 6 images arranged in a circular layout
- Navigation via previous/next buttons
- Keyboard support (ArrowLeft/ArrowRight)
- Click on any card to rotate the carousel
- Active card highlighting with scale and glow effects
- Smooth CSS transitions

## Usage

Open `index.html` in a web browser to view the gallery.

## Controls

- **Previous/Next buttons**: Navigate through images
- **Arrow keys**: Left/Right arrows to rotate
- **Click cards**: Rotate to next image

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and 3D transforms
- `app.js` - Carousel logic and event handlers
- `images/` - Sample images
